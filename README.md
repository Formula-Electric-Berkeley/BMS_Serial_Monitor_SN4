# Serial Monitor

## Notes

Design inspired by CoolTerm serial monitor app on MacOS.

## Serial Input

Cell Data
- cell
- \<int cell>
- \<int bank>
- \<float voltage>
- \<float temperature>
- \<bool balance>

Relay Data
- relay
- \<bool AIR+ open>
- \<bool AIR- open>
- \<bool precharge open>

IVT Data
- ivt
- \<float current>
- \<float voltage 1>
- \<float voltage 2>
- \<float voltage 3>
